## âš™ï¸ **Installation & Setup**

### **1. Clone the Repository**

```sh
git clone https://github.com/badhon252/The-Social-Chamber
cd The-Social-Chamber
```

### **2. Install Dependencies**

```sh
npm install
```

### **3. Environment Variables**

Create a `.env.local` file and include values such as:

```
NEXTAUTH_SECRET=
NEXTAUTH_URL=
DATABASE_URL=
STRIPE_SECRET_KEY=
STRIPE_PUBLIC_KEY=
CLOUDINARY_API_KEY=
CLOUDINARY_SECRET=
CLOUDINARY_CLOUD_NAME=
```

### **4. Run the Development Server**

```sh
npm run dev
```

### **5. Build for Production**

```sh
npm run build
npm start
```
